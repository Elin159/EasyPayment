# 环境要求
- php版本:>=5.6
- laravel版本: Laravel5.5+

# 安装
```php
composer require zwxhenu/easy-payment

```
# 当前支持功能

## 微信支付
   - JSAPI支付
   - Native支付
   - 支付查询
   - 金额退款
   - 退款结果查询
## 支付宝
   - 网页版支付
   - 支付查询
   - 即时有密退款
## 百度钱包 
   - 网页版支付
   - 支付查询
   - 退款
   - 查询退款结果
